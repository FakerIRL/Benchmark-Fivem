# Benchmark-Fivem
- FiveM Benchmark Script
- Overview

This resource is a performance benchmark tool for FiveM designed to test client stability, FPS, and overall graphical performance under different conditions.

It simulates various in-game scenarios such as weather changes, time cycles, and visual effects to evaluate how well a system handles intensive gameplay situations.

- Features
Dynamic weather transitions
Day & night cycle simulation
Particle effects and animations
FPS monitoring
Performance stress test sequences
Simple and lightweight setup
- Installation
Download or clone this repository
Place the folder into your resources directory
Add the resource to your server.cfg:
ensure benchmark
Restart your server
- Usage
Join your FiveM server
Use the command:
/benchmark
The test will automatically run through multiple scenarios
